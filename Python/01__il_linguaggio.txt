Python è un linguaggio di programmazione ad alto livello, rilasciato
pubblicamente per la prima volta nel 1991 dal suo creatore Guido van Rossum.
Python è semplice da usare e imparare, è nato per essere
un linguaggio immediatamente intuibile.
Python è open source, può essere scaricato gratuitamente e usato a piacere
nelle proprie applicazioni, oltre a questo è possibile modificarlo
e ridistribuirlo.
Python è un linguaggio portabile scritto in C.
Si può usare su qualsiasi sistema opearivo, purché sia installato l'interprete.
Questo linguaggio è utilizzato in moltissimi campi:
script d'automazione, sviluppo web, sviluppo mobile, interfacce grafiche,
programmazione di rete, database, sviluppo videogame, sviluppo mobile,
ed eccelle nelle applicazioni scentifiche, come il data science,
machine learning, deep learning e intelligenza artificiale.
